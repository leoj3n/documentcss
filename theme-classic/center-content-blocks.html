Center content blocks

  • styles
center-content-blocks  

Set an element to display: block and center via margin. Available as a mixin and class.

<div class="center-block">...</div>

// Class
.center-block {
  display: block;
  margin-left: auto;
  margin-right: auto;
}

// Usage as a mixin
.element {
  .center-block();
}
